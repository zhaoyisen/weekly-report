# 智能周报 Smart Weekly Report 1.0.0 正式发布

智能周报是一款 Windows 桌面工具，用于快速记录日常工作事项，并通过用户自定义 AI API 将口语化记录优化为可用于周报的任务条目。

## 主要功能

- 快速记录工作事项
- 按日、周、月、年管理任务
- 自定义兼容 OpenAI Chat Completions 的 AI API
- 单条和批量 AI 优化
- 周报生成、复制和 Markdown 导出
- 本地数据存储、备份和恢复
- 系统托盘和 Ctrl+Alt+W 快捷唤起

## 发布文件

- `Smart-Weekly-Report-Setup-1.0.0-x64.exe`：Windows 安装包
- `Smart-Weekly-Report-Portable-1.0.0-x64.exe`：便携版

## 说明

- 当前版本暂未进行代码签名，Windows 可能提示未知发布者。
- 应用不会上传全部历史记录。只有在用户主动使用 AI 优化功能时，当前任务内容会发送到用户自行配置的 AI API 服务。
