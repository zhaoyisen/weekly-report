# 智能周报隐私与数据安全说明

版本：1.0.0

## 数据存储

智能周报默认将任务记录、项目、设置和周报相关数据保存在用户本地电脑。

应用不会主动上传全部历史记录，也不会将本地数据库同步到云端。

用户可以在应用的“设置 - 数据管理”中查看本地数据库位置，并手动备份或恢复数据。

## AI API 调用

智能周报支持用户自行配置 AI API。

只有在用户主动使用以下功能时，应用才会向用户配置的 AI API 服务发送当前待处理内容：

1. 单条任务 AI 优化。
2. 批量任务 AI 优化。
3. AI API 连接测试。

应用不会在后台自动上传全部任务记录。

## API Key 保存

用户填写的 API Key 会保存在本机。

应用优先使用 Electron safeStorage 调用系统安全能力进行加密保存。设置页默认只展示脱敏后的 API Key。

## 第三方服务

AI 请求发送到用户自行填写的 API Base URL。对应服务的数据处理方式、日志留存和隐私政策由该 AI 服务提供方负责。

请用户确认所配置的 AI 服务符合所在组织或个人的数据安全要求。

## 卸载与数据

卸载应用通常不会自动删除用户本地数据目录。

如果需要彻底删除数据，请先在应用设置中查看数据库位置，然后手动删除对应数据文件。
