# 更新日志

## 1.0.0 - 2026-06-15

首个正式版本。

### 新增

- Windows 桌面应用。
- 快速记录口语化工作事项。
- 日、周、月、年、全部记录视图。
- 当前周期任务统计。
- 任务新增、编辑、删除、搜索、筛选。
- 任务多选、批量 AI 优化、复制所选、批量删除。
- 项目管理。
- 自定义 AI API 配置。
- AI API 连接测试。
- 单条任务 AI 优化。
- 周报生成、复制、Markdown 导出。
- 本地 SQLite 数据库存储。
- 数据库备份与恢复。
- 系统托盘、关闭到托盘。
- `Ctrl+Alt+W` 快捷唤起并聚焦快速输入框。
- 正式应用图标。
- Windows 安装包和便携版。

### 说明

- 当前版本未内置 AI 服务，用户需要自行配置兼容 OpenAI Chat Completions 的 API。
- 当前安装包未代码签名，Windows 可能显示未知发布者提示。
